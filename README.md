# ToDo List 1.0

**ToDo List** на **Python** с **сохранением** данных в файл **JSON**.

## Описание

Простой ToDo List на **Python**, который **сохраняет** данные в файл **JSON** без использования **сторонних библиотек**.

Доступны:
* **русская версия** (`rus/main.py`);
* **английская версия** (`eng/main.py`).

Данные сохраняются в файл `data.json`.

## Требования

* Python 3.6 или выше.

## Функциональность

Программа позволяет:
* добавлять новые задачи;
* отмечать задачи как выполненные;
* удалять задачи;
* просматривать текущий список задач.

## Особенности

* Данные сохраняются в `data.json` между запусками.
* Доступны русская и английская версии интерфейса.
* Не требует установки сторонних библиотек.
